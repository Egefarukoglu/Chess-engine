A Python-based chess engine developed using object-oriented programming principles. The project focuses on implementing core chess mechanics and game logic.

## Features
- Board representation
- Legal move generation
- Piece movement rules
- Basic validation (check, legal moves, etc.)
- Work in progress: AI and advanced game logic

## Technologies
- Python
- Object-Oriented Programming (OOP)

Currently under development
